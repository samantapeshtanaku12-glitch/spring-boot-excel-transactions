# Spring Boot Excel Transaction Manager

A Java Spring Boot application for importing Excel transaction files, storing the data in PostgreSQL, and displaying it through a web interface.

## Features

- Upload Excel (.xlsx/.xls) files
- Read data using Apache POI
- Store transactions in PostgreSQL
- Spring Data JPA integration
- REST API endpoints
- Display uploaded transactions in a dynamic HTML table

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Apache POI
- HTML
- CSS
- JavaScript
